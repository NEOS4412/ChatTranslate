# 资产阶级与空间 *

我首先要说明，我不是以法国大学代表的身份发言，尽管我在因1968年事件而闻名的巴黎南泰尔大学任教社会学。顺便提一下，南泰尔大学的社会学学生曾接受过马克思主义教育。这种教育使他们认识到，不应该像其他国家的学生那样退缩到一个抗议性的微观社会中，而应该攻击整个资产阶级社会及其首脑——国家。我相信，这种马克思主义教育在1968年法国发生的事件进程中起到了一定作用。

我不是以法国大学成员的身份说话，也不是作为任何机构的代表，而是以我个人名义。我凭什么身份在这里向你们致辞？我是一名哲学家，但并非那种自诩“纯粹”并声称自足的哲学家。对我而言，哲学质疑社会和政治实践；它努力将这种实践提升到语言、概念和理论的层面上。在当前实践中，建筑、城市规划、住房问题和城市问题都至关重要。更进一步说：需要审视的是当前生产力的水平及其可能性。

简而言之，我是以马克思主义者的身份向你们致辞；

我想围绕发达国家的经验，针对现代资本主义提出一项理论分析，并以空间问题为核心。（很抱歉我不谈智利和拉丁美洲：我对它们不够了解。）我想回顾一下马克思的方法。他是从时代最突出、最普遍的特征入手，抓住最鲜明的特征，然后才去把握差异。他先立足于基本冲突，以此来确定和评估那些次要的、派生的、因时而变的冲突。我采用这种方法也是恰当的，因为这次会议是一次国际会议。

用更简单的话来说，我想跟各位简要谈谈法国和欧洲，谈谈那里出现的新情况，以及我们可以从中吸取的一些教训。我把得出结论的任务留给你们去思考。我没有资格给任何人上课。

诸位已经知道，法国是一个发达的工业国家，也就是说，在资本主义生产方式框架内，其生产力已达到很高水平。然而，法国并不属于最大的工业国家之列。正因如此，一场（经济、技术、制度上的）现代化正在进行，这场现代化使得我们能够以法国为良好范例来研究和界定新资本主义，即组织型资本主义。我用的不是“有组织的资本主义”，诸位将会看到，这一区别至关重要。

法国也是一种帝国主义。它并非最强大的帝国主义。自20世纪初以来，法国帝国主义一直遭受着更强大帝国主义的冲击：英国、德国、美国。法国最显著的特征，是民主革命、伟大的法国大革命（及其成果：人权）与帝国主义之间的矛盾，即资产阶级作为精明而冷酷的统治阶级、警察国家、对法国工人的剥削以及对法国境内350万外国劳工的超剥削，这两者之间存在着深刻矛盾。1968年，这一系列矛盾爆发了出来。但法国现实还有另外一些方面：法国存在着一个 巨大的首都，对这个国家来说过于庞大：巴黎。首都吸引一切：人、头脑、财富。它是决策和舆论的中心。巴黎周围延伸着从属的、等级化的空间；这些空间既受巴黎支配，又被巴黎剥削。帝国主义法国失去了殖民地的同时，一种内部新殖民主义却已建立。如今的法国包含一些过度发达、过度工业化、过度城市化的区域。以及许多欠发达状况日益加剧的区域，尤其是在布列塔尼和南部地区。

现在我们来看看法国的住房问题。它远未在资本主义制度下得到彻底解决，正如恩格斯在一部著名著作中所说的那样。尤其对于外籍工人而言。大量贫民窟仍然存在，无论是在老城市中心还是在郊区。然而，法国工人阶级的相当一部分人住房条件相对较好；在郊区，他们住着独立的小房子（"别墅"），以及围绕着新城市和老城区周边建造的所谓"大型住宅区"（大型多层租赁住宅群）。在过去二十年加速的城市化进程中，大部分工人和中产阶级因此住进了相对可以接受的住所，但缺乏建筑或城市规划方面的创新。相反，城市的这种扩张伴随着建筑和城市环境的退化。人们被分散，尤其是工人，远离城市中心。主导这种城市扩张的是经济、社会和文化上的隔离。经济和生产力在数量上的增长并未带来社会发展，反而导致了社会生活的恶化。郊区是破碎的、乡村化的城市。这不仅仅关乎巴黎。社会的城市化伴随着城市生活的恶化：中心破碎，从此丧失社会生活——人们在空间中受到隔离分布。这里存在一种真正的矛盾。我称之为空间的矛盾。一方面， 统治阶级和国家加强了城市作为权力和政治决策中心的地位——另一方面，这一阶级及其国家的统治却使城市分崩离析。

正是考虑到这些郊区居民、隔离和孤立，我在一本书中谈到了“城市权利”。这不是法律意义上的权利，而是类似于著名的、构成民主基础的人权宣言中所规定的权利。这些权利从未在字面上完全实现，但人们总是参照它们来界定社会状况。人权宣言后来补充了妇女权利、儿童权利等。我提议在这一清单中增加“城市权利”，以及“差异权”，这是考虑到构成现代空间社会以及全球范围内社会的不同族群和不同民族。

现在我要谈谈法国日益加剧的不安情绪。在相对资本主义繁荣和生产增长（即所谓的消费社会）的表象下，存在着诸多不满；许多人开始谈论“生活质量”，无论是在反对派还是执政党中都是如此。有些人谈论空间质量；有些人甚至想要“改变生活”。

空间质量意味着什么？这一观念更新了关于建筑质量、建造质量的旧观念；它获得了另一层含义：即一种精心构建、复杂且成功的、简而言之是恰当的社会空间，而不仅仅是受技术和政治权力支配的空间；这将与“改变生活”相辅相成。在这种空间质量实现之前，弥漫着一种不安和焦虑，它们不再满足于旧有的自由人文主义。而现在有一条重大消息，我不确定它是否是好消息，有些人甚至认为它是坏消息。经济增长和社会发展不再能混为一谈。长期以来，人们一直将各个方向上的增长——人口、经济、技术——与质性的社会发展等同起来。人们将它们 混杂在一起，以为增长会带来发展，以为数量迟早会带来质量。然而，数量只有在被质疑、被悬置、经历危机之后，才能带来质量；也就是说，在临界点和临界阈限之后！增长意识形态已遭受致命打击；此前人们怀着顽固的乐观主义，相信生产力和生产率的无限增长：更多的汽车、更多的电视机、更多的洗衣机或计算器。人们怀着同样的乐观主义，认为这种经济增长迟早会满足所有需求：物质的和所谓的“精神”需求。这种增长本应由左派接力，以备资产阶级和执政党破产之需。人们仍然怀着同样的意识形态，相信大型企业有利可图，相信人口和技术增长有益无害。这一庞大的意识形态建构正在缓慢但确定无疑地坍塌。其后果是什么？其后果是城市生活的困扰、自然及其资源的破坏，以及各种阻滞社会发展的障碍——即使它们不妨碍经济增长，也会造成这种结果。

因此，近来——几个月或至多一两年——旧有的乐观主义已被一种千禧年主义、末日论的意识形态所取代。许多人都在问：我们能否撑到2000年！

在这些忧患之上，还要加上环境与污染的关切——这些虽是真实但次要的现象，却被用作转移视线、掩盖本质的障眼法。想想生产主义的危机吧，想想生产主义意识形态在“发达”国家行将消失的事实。这是一个矛盾的过程，但意义的转变、一种转折正在发生。为增长而增长，如今已成为过时的东西。与此方向转变或多或少相关的重大变革已经出现。

政治经济学？它无非是关于工业生产、增长和分配的理论。

以及产品的转化。经济学家构建了或多或少成功的增长模型，通过或多或少地协调不同部门、“充分就业”、货币平衡和扩张来实现增长。那么政治经济学现在变成了什么？它也经历着一场危机。增长模型逐渐失去信誉。政治经济学正在转变为一种空间的政治经济学。我强调这些词语。这不再是将抽象的经济模型（涉及物品、商品、消费品的生产）应用于地理空间或几何空间（一个空洞的中介）的问题。政治经济学变成了空间的政治经济学。这意味着什么？首先，是对土地和空间的动员，而过去我们谈论的是“不动产”财富。住房市场变得普遍化。空间，即容积，被处理成同质的，其各部分可比较，因此可交换。建筑不再是一个次要工业分支、一个从属的经济部门。它变成了一个首要部门。它不仅在反馈、平衡、刺激方面发挥作用，而且在当前的资本主义经济中扮演着刺激角色；它成为生产的一个基本分支。

请允许我强调这些方面的每一个。我说住房市场在主要资本主义国家变得普遍化；这意味着空间被处理成同质的。由谁处理？由建筑商、建筑师、开发商（我们这样称呼那些拥有资本并主动发起盈利性建筑的人）。被碎化的空间各部分变得可交换。空间从属于金钱和资本，导致了一种量化，这种量化从每个地块的货币评估和商业化扩展到整个空间。土地所有权，在法国以及欧洲其他地区起源于封建制度，曾因伟大的法国大革命（1789年）而作为封建所有权被动摇。自那以后，它作为农业土地的资本主义所有权重新形成。这一过程正在加速，并转变为整个空间的资本主义所有权。与此同时 « 不动产 »——我们这里还这么称呼——正在流动起来，也就是说变成动产财富，被卷入交换的洪流中，卷入货币和资本的涨落中。与此同时，使问题更加复杂的是，曾经充裕的可建设用地在中心周边变得稀缺。空间已成为新的稀缺资源之一。这种围绕中心被维持和利用的稀缺性，促成了强制性的投机。还需要补充的是，空间的稀缺伴随着曾经充裕的资源与商品（如水、空气，甚至光线）的日益稀缺吗？

当然，这样的分析应当考虑人口因素、技术因素以及各种现象。我们暂且把这些放在一边。我们得出了一个核心观点：资本主义通过征服和整合空间得以维持。空间早已不再是被动的地理环境或空洞的几何介质。它已经变成工具性的了。

当然，此刻我并不是在谈论宇航员；对星际空间的征服紧随对地球空间的征服而来；征服的冲动已经超越了地球的边界。让我们强调这一点以使其清晰：在先进的工业国家，资本主义首先是通过抢占前资本主义形态，并以其支撑点——大型工业企业——为基础整合这些形态的空间，从而维持自身。不存在、也从未存在过多种生产方式并存的情况，正如阿尔都塞的结构-功能主义学派所声称的那样。在资本主义内部，过去存在、现在仍然存在着与土地私有制相联系的、持久的前资本主义社会关系；这些前资本主义关系过去被、现在仍然被整合进资本主义中，始终以土地私有制作为手段和基础。以农业为例。在马克思的时代，只有英国拥有资本主义农业。今天，在法国及其他地方，在土地所有权的法律和实际基础上，农业生产已成为工业生产的一个部门，因此也是资本主义的。反过来，工业生产会考虑整个已开垦的土地，并适应经济资源……

为了满足这一整体的技术需求。农业的整合不仅仅是通过建立大型谷物种植或畜牧农场来实现的；它还通过许多面积虽小但需要大量投资的农场来完成：如果园、乳制品、蔬菜和时鲜产品、葡萄园等。

现在让我们来考察城市。城市也曾是一种历史现实，一种前资本主义的社会形态：想一想古代城邦和中世纪城市。资本主义攫取了城市，按照其经济、政治和“文化”需求改造历史城市。城市，或多或少地破碎为郊区、边缘地带和卫星城镇，同时成为权力中心和巨大利润的来源。城市集聚体已不再是企业间的间隙组织或大型企业的被动背景：它实际上成了这些企业的一部分：它提供这些企业不可或缺的多种服务、交通和分包业务。在现代城市中，存在着一种真正的对空间、交通手段、建筑物、道路和街道的生产性消费。大量劳动力被投入其中，其生产力与用于机器维护和供能的劳动力同样重要。在我看来，这可以说是当前资本主义繁荣的秘诀之一。投入空间生产与维护、交通以及众多被称为“服务”活动的劳动力，通常报酬低廉，且所投入资本的有机构成很低。这一切仍然是在土地所有权的法律和实际基础上进行的。与此同时，历史城市遗留下来的部分正在退化；对历史空间的消费对应着资本主义空间的生产，但资本主义由此摧毁了它自身的条件，即作为决策中心的城市。这就是我所说的空间的矛盾。

农业加上城市？这已然是整个空间。关于空间征服与生产的这一论点，可以通过对生产力（用马克思主义的语言来说）的分析来加以检验。这些生产力不再依附于 在确定且孤立的场所中：企业，尽管这种定位仍具有重要性。生产力涉及整个空间：各种流、能源、原材料、信息、产品流通网络。

如果我们愿意，请提出这个问题：“资本主义是如何在大型工业国家中维持下来的，尽管工人阶级曾威胁它，尽管危机和战争（由它引发）倾向于摧毁它？”换句话说，用清晰的马克思主义术语：“社会生产关系的再生产是如何完成并持续进行的？”我说的是：社会生产关系的再生产，而非物质和人力的生产资料的再生产。这是一个关键问题，自马克思在《资本论》未发表章节中预见到这一问题以来，它被更明确地提出。这个问题包含了接下来的问题：“这些社会关系何时以及如何停止再生产？何时以及如何产生新的社会关系？” 对于这些问题，已经有了众所周知的答案。一些人仍然认为，资本主义是靠纯粹的意识形态压力维持的，即某些人所谓的国家意识形态机器。另一些人则认为，新的生产关系通过政治途径开创，并由政治手段构成。这些推理在我看来都不令人满意，我请求大家思考一下。依我之见，资本主义中的社会关系，即剥削与统治的关系，是通过并存在于整个空间、通过并存在于工具性空间之中得以维持的。我们还将回到这一点。至于政治本身，它从未构成并维持过除政治关系以外的任何东西，也从未构成并维持过社会关系；这正是马克思对黑格尔国家哲学以及黑格尔关于政治阶级的论点进行批判的本质。

让我们将考察局限于资本主义。它不仅通过将已有空间纳入其扩展范围而整合了该空间；它还通过将其纳入其扩张范围而确立了新的领域。休闲在欧洲以及先进的大型工业国家中已成为一项首要产业。

重要。我们为休闲征服了海洋、山脉甚至沙漠。休闲产业与建筑业相结合，将城市和城市化沿着海岸和山区延伸。此时此刻，我正在与马里奥·加维里亚一起研究西班牙的这些现象，调查还将继续在其他国家进行。这种休闲产业扩展到农业和传统工业生产未占据的空间。对于建筑师和城市规划师来说，它具有尚待发掘的价值。休闲对空间有一定的品质要求。数千万欧洲人，包括许多工人，从欧洲北部向南迁移到西班牙、意大利、法国南部。可以说，他们离开了消费空间，即资本主义财富的集中地（伦敦、汉堡、巴黎等），去消费空间：海滩、大海、阳光、雪……这种品质要求对建筑师和城市规划师产生了一个有趣的结果：似乎正是在休闲城市中，现代建筑和城市规划才产生了一些新东西；不多，且多半是讽刺性的而非成功的，但在这些地方有一种对空间进行品质占有的尝试，与无处不在的量化特征相冲突。

历史城市的瓦解实际上伴随着普遍的城市化，这仅仅是历史城市的退化，可以称之为它的“乡村化”。普遍的城市化不仅使所谓的房地产投机普遍化，不仅伴随了空间的商品化，而且导致了城市与乡村关系的混乱，形成了一种城市与乡村混乱混杂的熔岩、混沌，而非超越。因此，对地球空间的征服确实在实现，但却是从瓦解中的财产出发的。

这种状况产生了一个后果，或者说一个非常重要的含义：一种新的规划形式——空间规划。这种规划形式超越了旧的规划方法。

通过物质和物质平衡表；它超越了基于财务平衡的规划技术；它整合为更全面的战略和实践。这一做法在法国已相当深入。人们以相当精确的方式计算原材料和能源的路径与流量。信息机器还能预测产品的疏散网络。经过非常深入的运筹学研究后，企业被选址定位，例如在马赛附近的滨海福斯，或者北海边的敦刻尔克。更重要的是，工人阶级被空间性地分配；由此以一种至今难以想象的方式被操控。我可以展示剩余价值的机制和路径如何变得复杂化。剩余价值不再在其生产地实现，而是根据经济、金融和政治力量的战略在全球范围内分配。生产、实现和分配剩余价值因而涉及整个地球空间。

这恰恰是一种空间政治经济学。关于在空间中生产物品的问题并没有消失，远非如此，但问题发生了转移。我们从在空间中生产物品转向生产地球空间，后者包含前者、以前者为前提。我们从传统的工业选址地点考虑转向整个空间。结果是，空间变得具有战略性。所谓战略，指的是某种政治统治下的特定空间内所有资源被用作手段，以瞄准并实现全球乃至超越全球范围的目标。全球性战略同时是经济的、科学的、文化的、军事的和政治的。

过去，人们与马克思一样认为，资本主义将会撞上生产关系与所有制关系对生产力的阻碍。人们认为资本家无法保障经济增长，也无法为工业生产提供哪怕最低限度的组织。由马克思主义所启发的思想和行动，旨在从资产阶级手中夺取工业生产以确保增长。

今天，资产阶级当然未能成功建立一个全球范围内连贯的资本主义社会，即组织化的资本主义；然而，他们确实建立了一种组织化的资本主义，这种资本主义在很大程度上成功地支配并维持了一个全球市场——尽管脆弱、在货币方面受到威胁，却始终存在，甚至具有影响力。

作为其行动的基础，资产阶级和资本主义拥有一种内生于生产关系中的财产形式：土地所有权。这种所有权远未对资本主义框架内的增长设置障碍，反而成为其支撑点，然而它却将这种社会投向一种空间混乱，我恳请各位对此加以思考。

统治阶级如今将空间当作一种工具来使用。这是一种多用途的工具：分散工人阶级，将他们分配到指定地点——组织各种流动，使其服从于制度性规则——因此将空间从属于权力——控制空间，并以技术官僚的方式管理整个社会，同时维持资本主义生产关系。

统治阶级是否做到了这一点？我说没有，因为出现了新的矛盾。这些就是我所说的空间矛盾，它们已不再完全是马克思在黑格尔之后所分析的历史时间矛盾。在整体处理空间的技术能力与将空间分割成小块进行买卖交换之间存在着矛盾。这正体现了当前生产力与财产关系之间的矛盾形式。另一个人们才刚刚开始发现的空间矛盾，是一方的运动、流动、短暂性与另一方所寻求的固定、稳定、平衡之间的矛盾。

我们能否从统治阶级手中夺走这个工具：空间？也许可以，但条件是要清楚而公开地提出这个问题，即根据新的现实，而非根据一个多世纪前提出的工业生产问题。我再说一遍，也许可以，但条件是必须据此制定相应的战略。
